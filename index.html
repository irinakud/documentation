Introduction

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior.

"HyperText" refers to links that connect web pages to one another, either within a single website or between websites.

HTML uses "markup" to create structure for the content to display in a Web browser. HTML markup includes special "elements" or "tags" such as <head>, <title>, <body>, <header>, <footer>, <section>, <p>, <nav> and many others.

Web Page Structure

A basic web page structure is the following:

<!DOCTYPE html>
<html>
<head>
<!--metadata elements-->
</head>
<body>
<!--page content-->
</body>
</html>

Metadata elements are the following: <link>, <meta>, <title>, <style>.

Page content can be marked up using a variety of HTML tags, including <main>, <header>, <footer>, <nav>, <section>, <article>, <div>.

CSS Rules

CSS describes how HTML elements should be displayed. To select the elements for styling, we can use the following selectors:

  • element selects all <elements>. For example, selector p selects all elements <p>.
  • .myClass selects all elements with class="myClass".
  • #myId selects all elements with id="myId".
  • element.myClass selects all <elements> with class="myClass". For example, p.myClass selects all elements <p class="myClass">.
  • * selects all elements.

With CSS we can define colour of elements, their style, size, position, set a backround, a layout of elements, create a picture or an animation, specify how elements behave under different curcumstances (e.g. when a cursor hovers over them, when they are active/not active or when the web page changes it's size).

Accessibility

Accessibility in web design is when the UI and the content is easily understood by all people, including those with auditory and visual disabilities.

Therefore, having an accessible design means:

  1. Having a well-organized code with appropriate markup.
  2. Having text alternatives for non-text and visual content.
  3. Creating an easily-navigated page that's keyboard-friendly.

CSS Flexbox

CSS3 introduced Flexbox to create dynamic page layouts where elements are arranged in a predictable way for different screen sizes and browsers.

Flexbox is a one-dimensional layout as it has either a row or a column at a time.

When we assign style display: flex; to an element, we declare this element to be a flexbox (with one row by default). The flexbox direction can be specified by the flex-direction property, which has 4 values: row, row-reverse, column, column-reverse.

CSS Grid

CSS Grid Layout is the two-dimensional model with columns and rows together.

To declare that an element is a grid, we set display: grid; in the element style.

To add some columns to the grid, use the grid-template-columns property:

.myGridElement {
display: grid;
grid-template-columns: 50px 50px;
}

The number of parameters given to the grid-template-columns property indicates the number of columns in the grid, and the value of each parameter indicates the width of each column.

Other properties like grid-template-areas, grid-row-gap, grid-template-rows can be used for laying out the grid as well.
